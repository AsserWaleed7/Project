اللغات المتاحة

HTML

HTML هي لغة تُستخدم لتصميم هيكل صفحات الويب. تعتمد على الوسوم لتحديد العناصر مثل النصوص والصور. تُستخدم مع CSS وJavaScript لتحسين الشكل والتفاعل.

الذهاب الى الدورة
CSS

CSS هي لغة تُستخدم لتنسيق وتصميم صفحات الويب. تتحكم في الألوان والخطوط والتخطيطات لجعل الصفحة جذابة. تعمل بجانب HTML لتحديد الشكل بدقة وسهولة.

الذهاب الى الدورة
JavaScript

JavaScript هي لغة برمجة تضيف التفاعل لصفحات الويب. تُستخدم لإنشاء ميزات مثل الأزرار التفاعلية والنماذج الديناميكية. تعمل مع HTML وCSS لتطوير واجهات قوية ومرنة.
